# UI Screenshot Gallery

This folder contains screenshots of all major UI views for design review.

## Screenshot Naming Convention

Files are named using the format: `NN-viewname-mode-viewport.png`

- **NN**: Sequential number for ordering
- **viewname**: The UI view (dashboard, project-list, project-detail, feature-detail, task-detail, inbox, settings)
- **mode**: Light or dark theme
- **viewport**: MacBook Air (1366x768) or iPad (768x1024)

## Views Included

| View | Description |
|------|-------------|
| Dashboard | Main dashboard with stats and project list |
| Project List | Grid view of all projects |
| Project Detail | Detailed view of a specific project |
| Feature Detail | Detailed view of a feature with tasks |
| Task Detail | Detailed view of an individual task |
| Inbox | Inbox for capturing ideas, bugs, and feature requests |
| Settings | Application settings page |

## Viewports

| Viewport | Dimensions | Device Type |
|----------|------------|-------------|
| macbook-air | 1366x768 | Laptop |
| ipad | 768x1024 | Tablet |
